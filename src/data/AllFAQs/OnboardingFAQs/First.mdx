To set up the agnoStack Omni-Channel eCommerce Support plugin for Zendesk, please follow the steps below to install and configuration the plugin into your Zendesk subdomain.

<br/>
<br/>

## Install from Zendesk Marketplace

Go to the agnoStack listing in the <Anchor isExternal href={props.process.env.GATSBY_URL_ZENDESK_LISTING} target="_blank">Zendesk Marketplace</Anchor> and click the __Install__ button.

<br/>

<center>
  <img
    class="border"
    width="75%"
    src="/images/onboarding-zendesk-marketplace-screenshot.png"
    data-canonical-src="/images/onboarding-zendesk-marketplace-screenshot.png"
  />
</center>

<br/>

You will then be prompted with a modal window to "`Select account to install this app`" where you'll select your Zendesk subdomain and then click __Install__ to confirm.

<br/>

<center>
  <img
    class="border"
    width="50%"
    src="/images/onboarding-zendesk-oauth-screenshot.png"
    data-canonical-src="/images/onboarding-zendesk-oauth-screenshot.png"
  />
</center>

<br/>
<br/>

## Confirm Plugin Installation

You will be taken to Zendesk's "`Application Settings`" page (below). You should not have to enter in any data in this page.

<br/>

You can optionally change the application title (or if you have an existing Custom Field you are using for Order ID that you'd like to continue to make use of) as well as install the application to a subset of agents using standard "`Enable role/group restrictions`".

<br/>

<center>
  <img
    class="border"
    width="50%"
    src="/images/onboarding-zendesk-application-settings-screenshot.png"
    data-canonical-src="/images/onboarding-zendesk-application-settings-screenshot.png"
  />
</center>

<br/>

After you click the __Install__ (or __Install Another__) and awaiting the installation processing, you will remain on the __Settings__ > __Apps__ > __Manage__ page.

<br/>

<center>
  <img
    class="border"
    width="75%"
    src="/images/onboarding-zendesk-application-installed-screenshot.png"
    data-canonical-src="/images/onboarding-zendesk-application-installed-screenshot.png"
  />
</center>

<br/>
<br/>

## Complete Account Configuration

In order to complete the plugin Configuration, please open any Ticket in the system (__Home__ > __(Ticket)__) where you will then see the agnoStack plugin installed but awaiting our simple three step configuration and setup to your Commerce Provider.

<br/>

<center>
  <img
    class="border"
    width="75%"
    src="/images/onboarding-zendesk-agnostack-setup-screenshot.png"
    data-canonical-src="/images/onboarding-zendesk-agnostack-setup-screenshot.png"
  />
</center>

<br/>

```
(NOTE: You can revisit your configuration any time through the Manage Your Account link at the bottom of the plugin (available for any admin).)
```
